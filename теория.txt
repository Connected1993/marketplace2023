1) Класс - чертеж или шаблон (инструкция по созданию чего либо,в нем могут быть описаны базовые вещи )
2) Объект -  предмет который был создан по инструкции класса
3) $db = new PDO(....параметры....) - new  создается экземпляр класса с названием PDO
example 
$db->query('SELECT * FROM Users')

4)this - ссылка на обьект или контекст
5)self - ссылка на сам класс   self::Users
6)static - ссылка на статический класс, метод или свойство
7)свойство - переменная
8)метод - функция
9)статический метод - не нужно создавать обьект для вызова этого метода
10)динамический метод - нужен объект для вызова
